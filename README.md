## Deskripsi Proyek

Portofolio ini merupakan aplikasi web yang memudahkan kamu untuk memamerkan profil, riwayat pendidikan, pengalaman kerja, proyek, serta keahlian dengan tampilan modern dan responsif. Sangat cocok digunakan oleh individu seperti programmer, designer, dan profesional lain untuk menunjang personal branding.

## Fitur Utama

- **Profil Personal/Digital CV:** Data diri, foto, kontak, dan summary

## Instalasi & Setup

### Prasyarat

- Node.js (>= versi minimal)
- npm/yarn/pnpm

### Langkah Instalasi

1. **Clone repository ini:**
   ```bash
   git clone https://github.com/jwjooth/portofolio.git
   cd portofolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # atau
   yarn install
   ```

3. **Jalankan project secara lokal:**
   ```bash
   npm run start
   # atau
   yarn start
   ```
   Akses aplikasi pada `http://localhost:3000` (atau port sesuai konfigurasi).
